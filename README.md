# Mama's Table

A small Vue 3 + Vite app (frontend) for the Mama's Table project.

This repository contains the Vite-based app in the `Mamas_Table` folder.

## Quick start

Prerequisites: Node.js (see engines in Mamas_Table/package.json).

```bash
cd Mamas_Table
npm install
npm run dev
```

Open http://localhost:5173 in your browser (Vite default).

## Build for production

```bash
cd Mamas_Table
npm run build
# preview the production build
npm run preview
```

## Project structure

- `Mamas_Table/` — main frontend app (Vite + Vue 3)
  - `index.html` — app entry
  - `src/` — Vue single-file components and app code
  - `public/` — static assets
  - `package.json` — app scripts and deps (`dev`, `build`, `preview`)

- `package.json` — repo-level dependencies (minimal)
- `LICENSE` — project license
